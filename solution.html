Solutions!

1. Soundproof Walls

We can use soundproof equipment, like they use on walls. This can be made out of foam, and can be surrounding the engine/propellor. I reduce the noise being emitted because the walls are absorbing the sound.

2. Quiet Zones

We can establish quiet zones where ships should not cross. This can be a spot for marine wildlife to live without being disturbed.

3. More blades to the propellers

Each blade does some work. That is a law of physics. The fewer blades, the more work each does, so they produce more noise. More blades can reduce the noise emitted because each blade does less work. We can use more blades on ship propellors to make less noise.

4. Bubble Curtain!

We can make a motor that will shoot air upward, making miniscule bubbles. Believe it or not, this actually provides as a sound barrier. So animals don't get scared, and for extra protection, we can add kelp and other sea plants around the curtain. We can use this when construction happens offshore.


Now the marine animals can live in peace :D
